# Телеграм-бот «Московский культурный путеводитель»

Учебный бот-справочник по культуре Москвы: компактные тексты, подборки музеев и маршрутов, карточки с фотографиями.

## Быстрый старт
1) Установите зависимости (лучше в виртуальном окружении):
   ```bash
   python -m venv .venv && source .venv/bin/activate
   pip install -r requirements.txt
   ```
2) Укажите токен бота:
   ```bash
   export BOT_TOKEN=ваш_токен_от_@BotFather
   ```
3) Запустите:
   ```bash
   python bot.py
   ```

## Что умеет бот
- `/start` или `/menu` — главное меню.
- «Узнать о Москве» — текстовый ввод + фото.
- «Московские художники» — текст о Пименове, Плавинском, Чтаке + медиа-группы с их работами.
- «Путеводитель» — четыре подраздела с фото и кнопками мест:
  - Классика и реализм (Третьяковка, ГМИИ Пушкина).
  - Русский авангард и модернизм (Шаболовка, Центр авангарда, Новая Третьяковка).
  - Советское искусство и соцреализм (Новая Третьяковка, ВМДИ, мозаики).
  - Современное искусство (MMOMA, Гараж, МАММ, Винзавод).
- «Авторы» — подпись команды.

## Как адаптировать под другой регион
- Обновите тексты и кнопки в `build_main_keyboard` и хендлерах `info:...`, `guide:...`, `artists`, `authors`.
- Подложите свои изображения в папки `images/` и поменяйте пути в соответствующих словарях `GUIDE_*_IMAGES`.
- Подправьте приветствие (`START_MESSAGE`) и карточки разделов в обработчике `on_callback`.

## Проверка
- Локальный прогон: `python bot.py` (бот запустится в polling-режиме).
- Перед продакшеном используйте webhook и отдельный хостинг (Railway/Fly.io/VPS).
